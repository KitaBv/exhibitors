📱 Мобильное приложение

Установите портал как приложение на ваш смартфон для быстрого доступа без браузера.

🍎
iPhone / iPad (Safari)
  1. Откройте сайт в браузере Safari
  2. Нажмите кнопку «Поделиться» (↑)
  3. Выберите «На экран «Домой»»
  4. Нажмите «Добавить»
🤖
Android (Chrome)
  1. Откройте сайт в Chrome
  2. Нажмите «⋮» (три точки)
  3. Выберите «Добавить на гл. экран»
  4. Подтвердите установку
